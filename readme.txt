## Introduction
OSDMS is an online school document management system designed to help individuals and organizations efficiently organize, store, and manage documents and files. This repository contains the source code and documentation for OSDMS.

## Features
- Document Upload and Storage
- User Authentication
- Document Categorization and Tagging
- Search and Retrieval
- Access Control
- Version Control
